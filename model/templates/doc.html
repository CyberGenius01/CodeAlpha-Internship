Overview of Code 1: Machine Learning Models

Code 1 focuses on importing necessary libraries, setting up parameters, preprocessing data, training multiple machine learning models, and evaluating their performance. Here's a step-by-step breakdown:

Importing Libraries and Models

The code imports libraries like pandas, numpy, and various scikit-learn modules necessary for data preprocessing and model evaluation. It also imports several machine learning models.

Initialization of Parameters

Parameters such as the dataset path and columns to be removed are defined.

Importing Dataset

The Titanic dataset is loaded, and specific columns are removed. Missing values are handled using the SimpleImputer class.

Encoding Categorical Data

Categorical features are encoded using OneHotEncoder to convert text data into numerical form.

Normalization

Data normalization is performed using StandardScaler to standardize feature values.

Splitting the Dataset

The dataset is split into training and test sets using train_test_split.

Model Training

Six different machine learning models are trained, including:

  • Logistic Regression
  • K-Nearest Neighbors
  • Support Vector Classifier
  • Naive Bayes Classifier
  • Decision Tree Classifier
  • Random Forest Classifier

Prediction and ROC Curve Plotting

Predictions are made for each model, and ROC curves are plotted to compare model performance. The best model is selected based on ROC AUC scores.

Plotting Training Results

The decision boundaries of the best model are visualized using contour plots.
